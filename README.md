## 재고 증가, 감소 로직의 동시성이슈 해결방법
노션링크 : 
https://dynamic-cent-685.notion.site/116ed6ed7f7c806ba28ae070005f412e

### 전체 요약

- 재고시스템 수량 감소 로직에서 동시성 이슈 해결
- 여러 스레드가 동시에 재고감소 요청 시 결과 오류 (재고 100개에서 100의 스레드가 동시에 1개감소 요청 시  87개 남음)
- Synchronized 를 사용한 문제해결 → 좋은 해결책이라고 볼 수 없음
- Pessimistic Lock (비관락) 을 사용한 문제해결
- Optimistic Lock  (낙관락) 을 사용한 문제해결
- Redis 분산락을 사용한 문제 해결
